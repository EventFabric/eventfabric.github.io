Fields
------

Bold field names are required fields, others are optional.

.. table::

   ===============  ========  ======================================
   Field Name       Type      Description
   ===============  ========  ======================================
   **origin**       Lat/Lng   Lat/Lng where the arc starts
   **destination**  Lat/Lng   Lat/Lng where the arc ends
   color            Color     Arc color
   width            Number    Arc width
   ===============  ========  ======================================
