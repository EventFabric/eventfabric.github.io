Transformer¶

The transformer element receives an event, runs some logic with it and emits one or more events with the received value and any field added using specific blocks to add or modify fields.

The logic is done through a visual language.

Multiple Outputs¶

If a single event need to be splitted in multiple ones, you can create an _output list and append items to it.
